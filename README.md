# Emotion Detector 🎭

Emotion Detector is a web application that analyzes emotions from user-provided text using IBM Watson Natural Language Processing services.

The application detects emotions such as joy, sadness, anger, fear, and disgust based on the text entered by the user.

---

## Features

- Emotion detection from text
- IBM Watson NLP integration
- Flask backend
- Interactive frontend
- Real-time emotion analysis

---

## Technologies

### Backend
- Python
- Flask
- IBM Watson NLP

### Frontend
- HTML
- CSS
- JavaScript

---

## How It Works

1. The user enters text into the application
2. The frontend sends a request to the Flask backend
3. IBM Watson analyzes the text
4. The detected emotions are returned and displayed to the user

---

## Example

### Input

```text
I am really excited and happy today!
```

### Output

```text
Detected emotion: Joy
```

---

## Purpose

This project was created as part of an IBM course focused on Natural Language Processing and emotion analysis using Python and Flask.

The backend logic and emotion detection functionality were implemented in Python using Flask and IBM Watson NLP services, while the frontend components (HTML, CSS, and JavaScript) were provided by IBM as part of the course materials.
